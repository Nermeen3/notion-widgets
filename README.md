# notion-widgets
A collection of embeddable widgets for Notion.
Currency Converter
A minimal currency converter supporting GEL, USD, JOD, and EUR with live exchange rates.

Open Currency Converter -> https://nermeen3.github.io/notion-widgets/currency-converter.html
Open calculator         -> https://nermeen3.github.io/notion-widgets/calculator.html
# How to embed in Notion

Type /embed in your Notion page
Paste the link above
Resize to fit
